Постановка задачи

Ассоциативные массивы, или словари, представляют собой структуры данных, в которых значения хранятся в виде пар "ключ – значение". В языке программирования C# для создания таких структур используется класс Dictionary<TKey, TValue>.

Формат объявления словаря:

Dictionary<тип_ключа, тип_значения> имя_словаря = new Dictionary<тип_ключа, тип_значения>();

Пример объявления словаря, где ключи представлены целыми числами (int), а значения – строками (string):

Dictionary<int, string> My_dict1 = new Dictionary<int, string>();

Данный словарь позволяет хранить пары "целое число – строка", где каждое число выступает в роли уникального ключа, а соответствующая строка является значением.
